## How to Fork
Follow these steps carefully before starting your work.
1. Click the Fork button at the top right of this repository.
2. A copy of this repository will be created in your GitHub account.

## Fork Instructions

After forking this repository:

1. Enable GitHub Actions write permission by
Settings
→ Actions
→ General
→ Workflow permissions
→ Read and write permissions


## How to complete the assignment
1. Modify index.html
2. Update COLLABORATORS.md with your real name
3. Use commit message keywords
4. Run export-reflog.sh
5. Commit reflog evidence
6. Run test.sh
7. Score must be 100/100
